FORMATTING TIPS:
================

1. Always start each recipe with "RECIPE:" followed by the name
2. Use the exact labels: Type, Protein, Calories, Carbs, Fats, Ingredients
3. Numbers can include "g" or not (both "42g" and "42" work)
4. Type must be one of: Breakfast, Lunch, Dinner, Snack
5. Each recipe should be separated by a blank line
6. The parser is case-insensitive for labels
7. Order of fields doesn't matter (except recipe name must be first)


To use this template:
1. Copy the recipes above into Microsoft Word
2. Save as .docx or export as .pdf
3. In the app, go to Meal Library > Import Recipes
4. Select DOCX/PDF method
5. Upload your file
6. Click "Parse Document"
7. Preview and import!
